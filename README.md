# Python Bros

A simple Python console program for testing login, signup, and password reset features.

## Features

- Login test
- Sign up test
- Password reset with time limit
- Basic input validation
- Some fun hidden options

## How to run

Make sure you have Python installed.

Run:

python bros.py

## Menu

1. Login
2. Sign up
3. Reset password
4. Information
5. My fault
6. Some fun
7. clear

Type:

exit()

to quit the program.
